分贝运算 / 单位换算

声压级能量叠加与平均、背景噪声修正、声压/声强/声功率与分贝的双向换算

分贝叠加 / 平均

把多个声压级（dB）能量叠加或求能量平均。用于合并多个声源或多次测量。

— dB

L = 10·log₁₀(Σ 10^(Lᵢ/10))｜平均：除以个数后取对数。两个相等声源叠加 +3 dB。

背景噪声修正

从「声源+背景」总值中扣除背景噪声，得到声源本身的级。

声源净声压级
— dB

差值 <3 dB 时背景过高，结果不可靠（ISO 1996）；差值 >10 dB 可忽略修正。

声压 ↔ 分贝 (SPL)

基准声压 p₀ = 20 µPa。L_p = 20·log₁₀(p/20µPa)

声强 ↔ 声强级

声强 I = 单位面积通过的声功率（声强探头测得）。基准 I₀ = 1×10⁻¹² W/m²。L_I = 10·log₁₀(I/I₀)

声功率 ↔ 声功率级

声功率 W = 声源向外辐射的总功率。基准 W₀ = 1×10⁻¹² W。L_W = 10·log₁₀(W/W₀)

声强级 ↔ 声功率级（经测量面积）

声功率 = 声强 × 包络声源的测量面积 S，对应级的关系 L_W = L_I + 10·log₁₀(S)。声强法测声功率（GB/T 16404 / ISO 9614）正是在包络声源的测量面 S 上测平均声强，再乘面积得声功率。

声压级 ↔ 声功率级（含声场环境与距离）

点声源在不同声场环境下，距离 r 处声压级与声功率级的关系：L_p = L_W − 10·log₁₀(4πr²/Q)，Q 为指向性因数（声源所处空间决定）。对应 ISO 3744/3745 自由场/半自由场声功率测量模型。距离每增加一倍，自由场声压级降 6 dB。

此为自由声场点声源理想模型：仅含直达声、无反射混响。室内远处声压级受混响声场主导，需用房间常数另算（混响半径外趋于稳定，不再随距离下降）。全消声室最接近此模型；半消声室适用于置于硬反射地面的声源。

百分比 ↔ 分贝（幅值比）

用于增益/衰减比例换算。dB = 20·log₁₀(ratio)

DFAT 声场均匀性评估

直接场声学测试（Direct Field Acoustic Testing）用扬声器阵列在测试件（UUT）表面产生均匀、受控的直接声场，替代传统混响室声学测试。核心要求是测试区域内声压级均匀，行业典型容差为各控制点偏差 ±3dB 内（极差 ≤ 6dB），且需跨各倍频带分别满足。可选四种阵列布局：多点平面、弧形平面、平面三维、弧形三维。NASA-HDBK-7010 推荐用平面传声器阵列评估声场均匀性，按水平面/垂直面分别布置。

声学测量常用激励信号：正弦、白/粉/棕噪、对数/线性扫频、猝发音、MLS，含实时频谱示意

⚠ 信号源，非校准声级源。输出为数字电平（满量程 0 dBFS），实际声压取决于声卡、功放、扬声器与系统音量，未经校准。可作为外接测量系统的激励源，但 dB SPL 须用声级计读取。噪声信号听感比正弦更响，请从低音量起步并佩戴护听器。

实时频谱

播放时显示信号的频谱形状。白噪平直、粉噪 −3dB/oct 下斜、棕噪 −6dB/oct 更陡，扫频是一条移动的峰——一眼看出各信号的频率特性。

实时频谱峰值保持横轴：对数频率 20Hz–20kHz

频谱由 Web Audio AnalyserNode 实时分析。粉噪是声学测量（RTA、频响）的标准信号；对数扫频用于脉冲响应/RT60 测量；猝发音用于反射/回声分析。

相对响应测量（反向补偿）

按倒频带逐个播放固定电平的纯音，你用声级计在听音位置实测各频带 SPL 并填入，本工具自动算出相对响应曲线（各频带相对 1kHz 的偏差）。用于了解声卡→功放→扬声器→房间这条链路在测试点的实际相对响应，可发现哪些频段偏响/偏弱。

这是相对测量，不是绝对校准：结果反映"测试链路+房间+测点位置"的综合响应，换个位置或换个房间结果会不同；它不能反推出声卡的绝对 dBFS↔SPL 换算（那需要已知声级计读数对应的精确参考）。

隔声量评价

发声室/收声室多点声压级 → R / R′ / DnT → ISO 717-1 单值 Rw + C + Ctr

测量标准依据：实验室空气声隔声测量 GB/T 19889.3-2005 / ISO 10140-2；现场测量 GB/T 19889.4 / GB/T 17247.1 / ISO 16283-1；单值评价（Rw + C + Ctr）按 GB/T 50121 / ISO 717-1；门窗隔声分级 GB/T 8485；空气吸收修正 ISO 9613-1。隔声量定义 R = 10·log(入射声功率 / 透射声功率) dB。
注：GB 标准采用 1/3 倍频程、频率范围 125–4000 Hz；ISO 为 100–3150 Hz。本工具为工程估算，不替代具备计量资质的检测。

1选择隔声量类型

三种量共用同一组声压级数据，仅修正项不同。切换后下方结果实时重算。

2房间与试件参数

收声室吸声量 A 由各频带混响时间反算：A = 0.161·V/T（Sabine）。R 与 R′ 用 S/A，DnT 用 T/T₀（T₀=0.5 s）。

空气吸收主要影响 >1 kHz、大空间。修正后 A = 0.161·V/T − 4·m·V，m 为空气功率衰减系数，依温湿度气压算得（ISO 354 §8.1.2.3 思路，m 按 ISO 9613-1）。常规建筑量影响很小，默认关闭。
